# Mentoria Hub

EdTech-платформа для учеников 8–11 классов: поиск образовательных возможностей + асинхронные курсы Mentoria в одном месте.

MVP для хакатона Mentoria Hub (Working MVP Challenge).

## Что умеет платформа

- **Главная (Dashboard)** — «Продолжить обучение», статистика, рекомендации, ближайшие дедлайны
- **Каталог курсов** — фильтры по категории, уровню, языку, поиск и сортировка
- **Страница обучения** — видео-плеер, уроки, ключевые моменты, интерактивный квиз с мгновенной проверкой, прогресс-бар, сертификат
- **Возможности** — олимпиады, стипендии, стажировки, конкурсы с фильтрами и сохранением в избранное
- **Личный кабинет** — аватар, статистика, прогресс по предметам, бейджи-достижения
- **Практика** — мини-тесты по всем курсам с подсчётом баллов
- **Админ-панель** — добавление/удаление курсов и возможностей без пересборки сайта, список пользователей
- **Онбординг** — подбор курсов и возможностей под класс, интересы и цели ученика
- **Бонусы** — тёмная тема, мультиязычность (RU / EN / KZ), полная адаптивность под мобильные

## Технологии

| Слой | Технология |
|------|-----------|
| UI | React 18 + Vite |
| Стили | Tailwind CSS |
| Роутинг | React Router |
| Иконки | lucide-react |
| Данные | localStorage (без бэкенда — работает офлайн на демо) |

Рекомендации работают на простой логике: курсы и возможности помечены тегами интересов
(STEM, программирование, бизнес и т.д.), а онбординг сопоставляет их с выбором ученика.

## Запуск

```bash
npm install
npm run dev      # http://localhost:5173
```

## Сборка и деплой

```bash
npm run build    # собирает в dist/
npm run preview  # локальный предпросмотр сборки
```

Деплой на **Vercel**: импортируй репозиторий → framework определится как Vite → Deploy.
Либо **Netlify**: build command `npm run build`, publish directory `dist`.

## Что является тестовыми данными

- Курсы, уроки и квизы — в `src/data/courses.js`
- Возможности — в `src/data/opportunities.js`
- Пользователи в админ-панели — мок-список
- Прогресс ученика, избранное, профиль — сохраняются в localStorage браузера
